Library Management System (Python)

FEATURES:
- Add new books
- Register new users
- Borrow a book
- Return a book
- View borrowed books
- Search book by title
- Menu-driven interface

CLASSES:
1. Book
   - Stores book details
   - Checks availability
   - Updates quantity

2. User
   - Manages user information
   - Borrows books
   - Returns books
   - Displays borrowed books

3. Library
   - Manages all books and users
   - Adds books
   - Registers users
   - Searches books
   - Shows book list

HOW TO RUN:
1. Save the script as library.py
2. Run:
   python library.py
